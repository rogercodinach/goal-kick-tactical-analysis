# Goal Kick Tactical Analysis in Professional Football

## Overview
This project analyzes goal kick situations in professional football using
StatsBomb event data. The focus is on tactical decision-making during goal kicks,
applied to Bayer Leverkusen matches from the 2023/2024 season.

The project was developed as part of a Bachelor's Thesis in Sports Science
and Data Analysis.

## Data
- Source: StatsBomb Open Data
- Team: Bayer Leverkusen
- Season: 2023/2024

Due to licensing restrictions, raw data is not included in this repository.

## Methodology
- Event filtering and feature engineering focused on goal kicks
- Contextual variables (pressure, positioning, distance, pass type)
- Rule-based tactical model structured as:
  - WHEN: contextual conditions
  - HOW: execution type
- Scoring function to evaluate tactical decision quality
